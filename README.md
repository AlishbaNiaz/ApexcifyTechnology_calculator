# 🧮 Basic Calculator

A simple calculator built using **HTML, CSS, and JavaScript**.  
It supports all basic arithmetic operations (+, −, ×, ÷) with a clean and responsive user interface.  
Includes both button and keyboard input support.

---

## 🚀 Features
- Perform basic arithmetic operations:  
  ➕ Addition  
  ➖ Subtraction  
  ✖️ Multiplication  
  ➗ Division  
- Real-time result display  
- Clear button to reset screen  
- Keyboard support (press numbers, operators, Enter, Backspace)  
- Responsive and modern design

---

## 📂 Project Structure
```

calculator/
│-- index.html   # Main calculator file (contains HTML, CSS, and JavaScript)

```
---

## 🛠️ How to Run
1. Download or clone this repository.
2. Open `index.html` in any web browser.
3. Start calculating! 🎉

---

## 💻 Technologies Used
- **HTML** – Structure of the calculator  
- **CSS** – Styling and layout  
- **JavaScript** – Calculator logic and interactivity  

---

## 🎯 Bonus Features
- Handles invalid inputs (no double operators like `++` or `--`)  
- Stylish UI with hover effects and responsive layout  
- Works with both **buttons** and **keyboard**
